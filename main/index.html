  • How DocentPro Built a Multi-Agent Travel Companion with LangGraph

    ### DocentPro 使用 LangGraph 构建多智能体旅行助手 DocentPro 利用 **LangGraph** 和 **LangSmith** 开发了一个模块化、跨流程的 AI 旅行平台，将旅行规划从灵感发现到行程安排、实时对话整合为一站式体验。他们将系统拆分为景点、餐厅、酒店和活动四大领域，每个部分由独立的可复用智能体管理，确保一致性并提升开发效率。 为解决 LLM 生成内容不准确或路线不合理的问题，DocentPro 引入了地理聚类（K-Means）、路径优化、过滤无效地点等机制，结合 LLM 的创意与确定性逻辑，提高实用性。 通过 **LangSmith** 实现全流程可观测性，便于调试、监控用户交互，并快速迭代改进。此外，DocentPro 还利用 LangGraph 快速构建支持 12 种语言的音频导览系统，实现高效的内容生成、翻译和语音合成。 最终成果包括： - 可复用的领域智能体 - 混合 LLM 与算法的实用行程 - 全流程追踪与优化能力 - 多语言 AI 行程与音频导览 DocentPro 正持续推动 AI 在旅行领域的协作与创新。
