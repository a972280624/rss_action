### Harmonic 使用 LangGraph 和 LangSmith 构建投资代理

Harmonic 是一家初创公司发现引擎，通过聚合大量数据为风险投资家提供洞察和工具。其面临的问题是早期初创公司环境复杂，传统搜索方式耗时且难以找到理想目标。Harmonic 选择 LangGraph 和 LangSmith，利用自然语言搜索和优化能力，显著缩短用户查找目标初创公司的时间。

LangGraph Studio 的可视化调试功能使工程师能够快速跟踪状态并减少调试时间。LangGraph 的模块化框架还让 Harmonic 快速将代理工作流扩展到其他产品部分，例如将“研究代理”添加到每个公司资料中，节省投资者筛选和尽职调查的时间。

LangSmith 提供了强大的提示版本控制系统，支持团队协作优化提示，并通过跟踪模型调用和用户交互，帮助团队快速定位问题并进行数据驱动的调整。LangSmith 的集成还加速了 Harmonic 的迭代周期，确保任何更改都能经过严格测试。

最终，LangGraph 和 LangSmith 的使用显著提升了 Harmonic 的搜索和研究能力，搜索结果相关性更高，用户价值获取时间从小时缩短到不到一分钟，搜索成功率提高了 30%。

### Definely 用 LangGraph 构建多智能体 AI 系统

Definely 是一家专注于法律行业的公司，通过将 AI 驱动的解决方案集成到律师的日常工作流程中，提升效率和准确性。其目标是构建高效的法律工作基础设施，选择 LangGraph 来设计多智能体系统，帮助律师解决复杂法律任务。该系统包含多个专业智能体，可分解复杂任务、协同完成子任务，并动态调整策略，同时在关键时刻结合人类输入和审批。

LangGraph 的图结构架构使 Definely 能够创建高度可定制的多智能体框架，适应法律领域的复杂需求。其 AI 系统可自动提取合同关键信息、分析修改后果、回答复杂法律问题、起草和优化合同语言、无缝集成文档管理系统，并按照团队指南标记合同。这使得律师能够高效、准确地处理合同，同时保持对最终输出的完全控制。

未来，Definely 将继续扩展系统能力，优化智能体表现，并确保与更多法律数据库和工具的无缝集成，致力于打造提升生产力且保持高准确性的 AI 法律基础设施。
