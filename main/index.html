  • How DocentPro Built a Multi-Agent Travel Companion with LangGraph

    ### DocentPro 使用 LangGraph 构建多智能体旅行助手 DocentPro 利用 LangGraph 和 LangSmith 构建了一个模块化的多智能体系统，提升旅行规划体验。他们将旅行助手分为景点、餐厅、酒店和活动四个领域，每个部分由独立可复用的智能体管理，支持跨流程协作。该系统结合 LLM 的创造力与确定性算法（如 K-Means 聚类、路线优化）以生成实用行程，并通过 LangSmith 实现全流程追踪与调试。此外，DocentPro 还利用 LangGraph 快速构建了支持 12 种语言的音频导览系统，实现高效内容生成与全球化扩展。
