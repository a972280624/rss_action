  • How DocentPro Built a Multi-Agent Travel Companion with LangGraph

    ### DocentPro 使用 LangGraph 构建多智能体旅行助手 DocentPro 利用 LangGraph 和 LangSmith 构建了一个模块化多智能体系统，提供从旅行发现、规划到预订的完整体验。他们将旅行助手分解为景点、餐饮、酒店和活动四大领域，每个模块可复用并适用于不同场景，提升开发效率与一致性。 系统结合 LLM 的创造力与确定性算法，通过 K-Means 聚类、路线优化、过滤虚假信息等手段，确保推荐内容实用可靠。LangSmith 提供全流程可观测性，便于调试与优化，尤其在多智能体协作中至关重要。 此外，DocentPro 还利用 LangGraph 快速构建了支持 12 种语言的音频导览系统，实现内容生成、翻译与语音合成自动化。目前，该平台已支持全球旅客获取 AI 行程建议与多语言音频导览。
