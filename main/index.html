# LangSmith Alerts：提前捕获生产故障

可靠的应用程序是优质用户体验的基础，而提前捕获故障是关键。LangSmith 推出了Alerts功能，帮助开发者实时监控 LLM 应用和代理。支持基于错误率、运行延迟和反馈评分等关键指标设置告警。

为什么主动监控很重要？

LLM 应用面临两大挑战：

  1. 对外部服务的依赖：模型提供商、API、数据库等依赖的服务可能因中断或延迟影响体验。
  2. 质量和正确性：LLM 输出质量可能因提示、模型或输入变化而不稳定。

通过反馈评分（用户反馈或在线评估），告警功能可提前发现质量问题。

LangSmith 告警功能概览

支持以下指标：

  • 错误数量和比率
  • 平均延迟
  • 平均反馈评分

可以按模型、工具调用等过滤，并设置聚合窗口（5 或 15 分钟）和阈值。告警可通过 PagerDuty 或自定义 Webhook 集成到现有工作流中（如 Slack）。

立即开始设置告警

未来计划

未来将增加更多功能，例如：

  • 运行计数和 LLM Token 使用量告警
  • 相对值变化告警（如延迟增加 25%）
  • 自定义时间窗口告警

# Harmonic 的投资智能助手案例总结

Harmonic 是一家专注于初创企业发现和投资的引擎，通过整合公开数据和私有数据，为风险投资（VC）提供高效的工作流工具和洞察。借助 LangGraph 和 LangSmith，Harmonic 构建了一个智能投资助手，帮助 VCs 更高效地发现、筛选和赢得优质交易。

问题：发现最令人兴奋的初创公司

早期初创企业的复杂生态让 VCs 难以快速找到符合其投资目标的企业。传统的搜索方法虽然功能强大，但需要手动组合数百个过滤条件，耗时且低效。

解决方案

  1. 自然语言搜索：Harmonic 引入了基于自然语言的搜索功能，用户可以通过简单的描述快速获取精准的结果。
  2. LangGraph Studio：提供了模块化工作流和强大的调试工具，显著提升了开发效率，允许团队快速迭代复杂的智能代理。
  3. LangSmith：通过版本控制和性能跟踪优化了模型调优和协作流程，确保每次更改都能被量化评估。

影响与成果

  • 搜索时间从小时级缩短至分钟级，正向结果提升 30%。
  • 新增即时市场地图和深度研究能力，结合用户 CRM 数据和网络，进一步提升了投资决策效率。
  • 投资者能够更专注于创始人互动，而非繁琐的筛选过程。

Harmonic 的创新技术显著提高了 VC 的工作效率，帮助他们在竞争中占据优势。
