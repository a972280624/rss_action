构建可靠代理系统的核心挑战在于确保在每一步中，大型语言模型（LLM）都能获得适当的上下文。这包括精确控制输入LLM的内容，以及运行适当的步骤以生成相关内容。代理系统既包括工作流，也包括代理（以及两者之间的所有形态）。大多数代理框架并非声明式或命令式的编排框架，而只是一系列代理抽象。这些抽象虽然便于入门，但往往会掩盖LLM在每一步中获得适当上下文的复杂性。所有形态和大小的代理系统（无论是代理还是工作流）都能从框架提供的相同功能中受益，或者从头开始构建这些功能。LangGraph可以被视为一个具有声明式和命令式API的编排框架，并在其上构建了一系列代理抽象。

### 阿布扎比TAMM平台的AI助手：借助LangChain和LangGraph提升政务服务

阿布扎比政府的TAMM平台是一个一站式服务平台，提供超过940项服务，涵盖驾照申请、罚款缴纳和个人文件查询等。2024年10月，TAMM 3.0版本发布，引入了AI助手，全面嵌入服务交付、客户关怀、日常运营和决策制定等环节。TAMM的AI助手不仅回答问题，还能主动预测用户需求、执行个性化任务、自动化服务流程，并生成可操作的洞察，提升用户体验。

AI助手基于LangChain和LangGraph构建，具备五大关键工作流程：服务查询与信息检索、用户特定数据交互、服务执行、通用知识与讨论以及支持流程。这些流程确保了服务的可访问性、个性化和高效性。此外，AI助手还配备“拍照上报”功能，用户可通过拍照或录像提交问题，AI自动提取细节并提交报告，提升政府响应效率。

TAMM的AI助手采用多智能体架构，确保数据管理和流程处理的灵活性和标准化。未来，TAMM将继续引领数字政府服务的发展，为公民提供更高效、更智能的服务体验。

### Harmonic 如何利用 LangGraph 和 LangSmith 构建投资代理

Harmonic 是一家初创企业发现引擎，通过整合大量公开和私有数据，为风险投资家提供有价值的洞察和工作流工具。然而，传统搜索方式复杂且耗时，难以快速发现目标初创企业。为此，Harmonic 采用 LangGraph 和 LangSmith，实现了自然语言搜索和查询优化，显著提升了搜索效率。

LangGraph 的模块化框架和可视化工具帮助 Harmonic 快速构建和调试复杂的搜索代理，例如执行复杂查询：“显示过去一年在旧金山或纽约获得顶级投资者融资、与团队成员有联系但过去一年未联系的 AI 公司。”LangSmith 则提供了强大的提示版本管理和评估功能，支持团队协作优化模型和提示，确保性能提升。

通过 LangGraph 和 LangSmith，Harmonic 的搜索结果相关性大幅提升，用户获取价值的时间从小时缩短到不到一分钟，搜索成功率提高了 30%。此外，Harmonic 还新增了即时市场地图和研究功能，帮助投资者更高效地筛选、评估和赢得最佳投资机会。

### Definely：用LangGraph构建多智能体AI系统

Definely通过将AI驱动的解决方案集成到律师的日常工作中，正在革新法律行业。它专注于提高效率和准确性，正在构建可在Microsoft Word中运行的专用智能体，帮助律师简化合同起草、谈判和审查流程。为了解决合同起草、审查和谈判流程中的挑战，Definely正在构建一个由多个专用智能体组成的先进系统，能够分解复杂任务、动态调整方法，并在关键时刻纳入人工输入。

Definely选择LangGraph来构建其多智能体系统，因为其基于图的架构能够创建灵活且高度可定制的框架，适合法律领域复杂的需求。该系统可以提取合同关键信息、分析合同变更的影响、回答复杂法律问题、起草和增强合同语言，还能与律师事务所的文档管理系统集成，确保合同符合公司标准。

未来，Definely将继续扩展系统功能，优化智能体性能，并与更多法律数据库和工具无缝集成，致力于打造更高效、更准确的法律工作流程。
