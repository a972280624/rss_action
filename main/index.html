    # Jimdo案例总结 Jimdo是一家欧洲领先的网站构建平台，18年来为190个国家的自由职业者提供服务。为解决个体创业者在营销、销售等方面缺乏专业支持的问题，Jimdo基于LangChain生态系统构建了AI商业顾问“Jimdo Companion”。该系统包含两大核心：**Companion Assistant**（智能助手，能理解用户语调并提供任务指导）和**Companion Dashboard**（聚合多数据源，提供个性化业务洞察）。依托LangGraph.js和LangChain.js实现复杂工作流编排，并通过LangSmith保障输出质量。上线后，使用Companion的用户首月客户联系量提升50%，客户活动量增加40%。未来，Jimdo将向自动化执行演进，打造全流程AI商业平台。
