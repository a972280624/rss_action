    # Cisco Outshift 实现10倍生产力提升的AI平台工程师案例 Outshift是Cisco的创新孵化引擎，专注于物联网、量子计算和下一代基础设施等前沿技术。其平台工程团队为多个孵化项目提供基础服务，但复杂的云原生环境带来了诸多挑战，如响应延迟、认知负担过重和效率低下。 为此，Outshift开发了 **JARVIS AI平台工程师**，通过LangGraph构建多智能体系统，并结合LangSmith和agentevals优化性能。JARVIS的核心功能包括： 1. **知识管理**：整合文档、代码等数据，快速提取洞察。 2. **自助服务能力**：自动化处理CI/CD请求、资源分配等任务。 3. **代码生成**：将自然语言转化为Kubernetes配置。 JARVIS集成到多个开发者界面（如Jira、Backstage、Webex和CLI），显著提升了效率： - CI/CD设置从一周缩短至1小时。 - 资源分配从半天减少到秒级。 - 消除了开发者与平台团队间的冗余沟通。 ### 关键经验 1. 多智能体系统释放潜力。 2. 用户体验无缝集成至关重要。 3. 持续评估和基准测试确保可靠性。 Outshift正通过JARVIS推动AI在平台工程中的应用，开启人机协作的新未来。 [了解更多](https://outshift.cisco.com/?ref=blog.langchain.dev)
