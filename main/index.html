    # Cisco Outshift 的 JARVIS：提升平台工程生产力的 AI 解决方案 Outshift 是 Cisco 的创新孵化引擎，专注于物联网、量子计算和下一代基础设施等前沿技术。其平台工程团队通过构建基础服务加速各类孵化项目。然而，复杂的云原生环境和频繁的开发者请求导致团队面临效率低下、认知负荷过高和任务积压等问题。 为解决这些问题，Outshift 开发了 **JARVIS AI 平台工程师**，利用 LangGraph 构建多智能体系统，并集成 LangSmith 和 agentevals 进行追踪与评估。JARVIS 的核心功能包括： 1. **知识管理**：整合文档、政策和代码库，快速提取洞察。 2. **自助服务能力**：自动化处理 CI/CD 请求、资源分配等常见任务。 3. **代码生成**：通过自然语言输入生成 Kubernetes 配置。 JARVIS 被集成到 Jira、Backstage、Webex 和 CLI 等开发工具中，极大提升了开发者体验。其成果显著： - 一周的任务缩短至一小时以内。 - 资源调配从半天减少到秒级完成。 - 减少了开发者与平台团队之间的沟通成本。 ### 核心启示 - 多智能体系统的潜力巨大。 - 用户体验的无缝集成是关键。 - 持续评估和基准测试确保系统可靠性。 Outshift 正在引领 AI 在平台工程中的应用，未来将继续探索 AI 与人类协作的无限可能。 [了解更多](https://outshift.cisco.com/blog/topic/platform-engineering?ref=blog.langchain.dev)
