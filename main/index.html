  • Introducing DeepAgents CLI

    **DeepAgents CLI 发布：支持持久化记忆的智能代理** LangChain 推出 **DeepAgents CLI**，一款支持持久化记忆的智能代理工具，适用于编码、研究和自动化任务。它可直接在终端运行，具备读写文件、执行 shell 命令（需确认）、网页搜索、调用 API、跨会话记忆及任务规划等能力。默认使用 Anthropic Claude 模型，通过 `.env` 配置 API 密钥。安装后启动，即可让代理自动分析代码并提交修改建议。 其核心是**持久化记忆系统**，知识存储于 `~/.deepagents/` 目录，遵循“记忆优先”协议，可长期保存项目规范（如 API 设计），并在后续任务中复用。支持创建多个独立代理，便于多项目管理。详情见 [GitHub](https://github.com/langchain-ai/deepagents) 与 [文档](https://docs.langchain.com/oss/python/deepagents/overview)。
