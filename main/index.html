  • How DocentPro Built a Multi-Agent Travel Companion with LangGraph

    ### DocentPro 使用 LangGraph 构建多智能体旅行助手 DocentPro 构建了一个 AI 旅行平台，通过 **LangGraph** 和 **LangSmith** 创建模块化多智能体系统，提升旅行规划体验。该系统将旅行规划分为景点、餐厅、酒店和活动四大领域，各由独立可复用的智能体管理。这种方式避免重复开发，保持一致性，并便于测试优化。 系统结合 LLM 的创造力与确定性逻辑，利用 **K-Means 聚类**、**路线优化** 和 **过滤机制** 来生成现实可行的行程。**LangSmith** 提供全程可观测性，支持调试和行为分析，确保多智能体协作透明可靠。 此外，DocentPro 还构建了支持 **12 种语言的音频导览系统**，采用 LangGraph 的 map-reduce 工作流，实现内容研究、叙事生成、翻译和语音合成的自动化流程。最终输出结构化的多语言音频指南，全球扩展性强。 总结：DocentPro 打造了一个智能化、可追踪、易扩展的旅行助手系统，推动 AI 在旅游领域的协作与创新。
