  • How DocentPro Built a Multi-Agent Travel Companion with LangGraph

    ### DocentPro 使用 LangGraph 构建多智能体旅行助手 DocentPro 利用 **LangGraph** 和 **LangSmith** 开发了一个模块化多智能体系统，打造从旅行规划到实时对话的无缝体验。通过将旅行助手拆分为景点、餐厅、酒店和活动四大领域，每个部分由独立代理负责，实现逻辑复用与一致性。 系统结合 **LLM 的创造性** 与 **确定性算法的精确性**，引入地理聚类、路径优化、过滤闭店地点等机制，避免不现实的建议。同时，借助 **LangSmith** 提供全流程可观测性，便于调试和优化。 DocentPro 还利用 LangGraph 快速构建了支持 **12 种语言的音频导览系统**，通过研究 → 生成 → 翻译 → 文本转语音的工作流，实现高效内容生成与全球扩展。 最终成果：一个可追踪、易改进、支持 AI 行程规划与多语言音频导览的旅行平台。
