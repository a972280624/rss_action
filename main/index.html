    # Webtoon Entertainment 使用 LangGraph 构建智能工作流以扩展故事理解 **Webtoon Entertainment**（纳斯达克股票代码：WBTN）是全球数字娱乐公司，以垂直滚动漫画格式闻名，运营着 **WEBTOON** 和 **Wattpad** 两大平台，连接全球用户与多样化的内容。 为应对海量内容处理需求，Webtoon 开发了 **Webtoon Comprehension AI (WCAI)**，利用 **LangGraph** 的智能工作流技术，实现对角色、叙事和视觉/文本上下文的深度理解。WCAI 已部署于市场营销、翻译和推荐团队，通过自然语言查询提供即时洞察、摘要和角色亮点，大幅减少手动操作。 ## 技术细节 WCAI 基于混合架构，结合 **Vision-Language Models (VLMs)** 和 **LangGraph** 的节点化工作流，支持动态路由处理多种业务需求。核心工作流包括： 1. **角色识别**：提取角色名称、角色关系及代表图像。 2. **说话人识别**：分析对话气泡并将其与角色关联。 3. **叙事提取**：生成结构化摘要，捕捉关键情节和情感点。 4. **领域专家应用**：根据用户意图生成业务洞察，如营销主题匹配或推荐场景提取。 ## 为何选择 LangGraph？ - **可控性**：满足生产环境中对透明性和可靠性的高要求。 - **生产就绪**：轻松部署 API 并集成到现有系统，借助 **LangSmith** 提供强大的调试和监控能力。 ## 成果与展望 WCAI 显著减少了手动工作量（如关键词提取效率提升 70%），提升了团队生产力。未来计划包括： - 系统化评估工作流性能。 - 细化人机交互控制。 - 引入外部数据源增强分析能力。 通过 WCAI 和 LangGraph，Webtoon 正在定义叙事理解的未来，赋能创意团队并最大化 IP 潜力。
