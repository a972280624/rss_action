# LangSmith Alerts：提前捕获生产故障

可靠的应用程序是优质用户体验的基础，而提前捕获故障则是关键。LangSmith 推出了 Alerts 功能，帮助开发者实时监控 LLM 应用和代理。支持基于 错误率、运行延迟 和 反馈评分 等关键指标设置告警。

为什么主动监控很重要？

LLM 应用面临两大挑战：

  1. 对外部服务的依赖：模型提供商、API、数据库等依赖的中断或延迟会显著影响用户体验。
  2. 输出质量和正确性：提示词、模型或输入的小变化可能导致结果不可预测，反馈评分告警可提前预警质量问题。

LangSmith 告警功能概述

支持以下指标的告警：

  • 错误数量和比率
  • 平均延迟
  • 平均反馈评分

可通过筛选条件（如模型类型、工具调用）聚焦特定运行子集，并设置聚合窗口（5 或 15 分钟）和阈值。告警可通过 PagerDuty 或自定义 Webhook（如 Slack）集成到现有工作流中。

未来计划

  • 新增告警类型：运行计数和 LLM Token 使用量
  • 支持相对值变化告警（如延迟增加 25%）
  • 自定义时间窗口告警

# Harmonic 利用 LangGraph 和 LangSmith 构建投资智能助手

Harmonic 是一家专注于早期创业公司发现和分析的引擎，通过整合海量公共数据与私有数据，为风险投资人（VC）提供高效的筛选工具。传统上，用户需要在数百个字段中手动筛选目标公司，耗时且低效。为解决这一问题，Harmonic 借助 LangGraph 和 LangSmith 开发了基于自然语言搜索的投资助手，显著提升了用户体验。

核心挑战

VC 在早期阶段寻找潜力初创公司时面临复杂的数据筛选难题。过去，Harmonic 的强大搜索功能虽然支持多条件组合，但操作繁琐，难以快速找到最佳匹配。

解决方案

  1. LangGraph Studio
  • 提供可视化调试工具，工程师可直接检查节点状态并调整模型运行，无需重新执行整个流程。
  • 模块化设计使独立工作流（如“研究代理”）能快速复用到其他产品功能中，节省开发成本。
  • 支持复杂查询，如结合地理位置、融资历史、人脉关系等条件，精准定位目标公司。
  1. LangSmith
  • 提供强大的提示词版本管理和协作开发环境，优化团队对模型调优的效率。
  • 集成评估系统，确保每次更改都能通过预定义指标测试，提升迭代速度和可靠性。

成果与影响

  • 搜索时间从数小时缩短至 1 分钟以内，相关性提升 30%。
  • 新增即时市场地图和综合研究报告功能，帮助投资者更高效地筛选、评估和赢得优质交易。
  • 结合用户 CRM 数据和公开网络信息，进一步增强了洞察力和决策支持能力。

Harmonic 的创新实践展示了如何利用先进工具提升投资研究效率，为 VC 行业树立了新标杆。
