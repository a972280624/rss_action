  • How DocentPro Built a Multi-Agent Travel Companion with LangGraph

    DocentPro 利用 LangGraph 和 LangSmith 构建了一个模块化多智能体旅行助手平台，旨在为用户提供从探索、规划到预订的一站式旅行体验。该系统将旅行规划分解为景点、餐厅、酒店和活动四大领域，每个领域由独立的智能体管理，实现跨功能复用与一致性。 通过结合 LLM 的创造力与确定性算法（如 K-Means 聚类、路径优化），系统能生成既有趣又实用的行程安排，并过滤虚构或关闭的地点。LangSmith 提供全流程可视化与调试能力，增强系统可靠性。 此外，DocentPro 还实现了支持 12 种语言的音频导览系统，采用 Map-Reduce 式工作流，涵盖内容研究、叙事生成、翻译和语音合成。该系统现已全球部署，助力用户便捷出行。
