### 如何用 LangGraph 和 LangSmith 构建投资助手

Harmonic 是一家初创公司发现引擎，通过整合海量公开和私有数据，帮助风险投资家（VC）发现潜在初创公司并提供洞察和工作流工具。然而，传统的搜索方式复杂且耗时，用户难以快速找到符合投资主题的初创公司。为此，Harmonic 利用 LangGraph 和 LangSmith 构建了自然语言搜索和优化功能，显著提升了搜索效率。

LangGraph Studio 助力开发

Harmonic 选择 LangGraph 是因为其生态系统方法，允许将所有提示存储在 LangSmith 中，通过 LangChain 调用目标模型，并在 LangGraph 中构建可组合的工作流。LangGraph Studio 的可视化界面让工程师能够跟踪状态，直接链接到每个节点的 LLM 状态，大大减少了调试时间。

LangSmith 助力评估与协作

LangSmith 提供了强大的提示版本管理系统，支持团队协作优化提示。通过 LangSmith，Harmonic 能够快速管理数据集和评估，确保每次更改都能经过严格测试，从而加速迭代周期。

效果显著

实施 LangGraph 和 LangSmith 后，Harmonic 的搜索和研究能力大幅提升。用户获取价值的时间从数小时缩短到不到一分钟，搜索成功率提高了 30%。此外，Harmonic 还增加了新的功能，如即时市场地图和结合公共网络数据的研究能力，帮助投资者更好地筛选、评估和赢得最佳交易。
